*  run the script make frontend and backend started

## Api for CRUD
* Create:
  methode:post,url:api/product
* Read:
  methode:get,url:api/search/{keyword}
* Update:
  methode:put,url:api/product/{product_id}
* Delete
  methode:delete,url:api/product/{product_id},
